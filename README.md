# fastapidemo

# Setup development environment
Create Python virtual environment
```
mkdir .venv
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
Install Serverless
```
npm 

```
# Deployment
--------------------------------
```
sls deploy
```
# Test
--------------------------------
Test with Pydantic validation
- test_book.py
Run
```
pytest
```
